## Student Enrollment Dashboard
This Power BI dashboard visualizes student enrollment data from various colleges in Agra. It helps users understand overall student strength, compare colleges, and analyze gender distribution.

## Objective
The main aim of this dashboard is to provide a clean, data-driven view of student enrollments for better decision-making at the institutional or administrative level.

## Tech Stack: Excel, Python(Pandas), Power BI, Dax, Power Bi Query, Data Cleaning
